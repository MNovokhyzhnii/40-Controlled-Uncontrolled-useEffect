# React Forms Homework (Vite + React)

Додаток показує:

- **ControlledForm** — керування полями через `useState` + валідація.
- **UncontrolledForm** — читання значень через `useRef`.
- **DataFetcher** — запит до API (`useEffect`) + стани loading/error/success.

## Стек

Vite + React

## Локальний запуск

```bash
npm install
npm run dev
```
